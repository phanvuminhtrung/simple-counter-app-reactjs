# REACTJS COUNTER APP
- Simple project to use the ReactJS template
- Demonstrate the understanding of core concepts of React, such as components, state, and event handling

## RUN PROJECT
- cd counter-app
- npm start
- open browser and inspect: http://localhost:3000
